# student-performance-ml
Student Performance Classification (Machine Learning)
# Overview
  This project builds and evaluates machine learning models to predict whether a student will pass based on study habits and academic history.
# Objective
  To explore supervised learning techniques including logistic regression and decision trees for binary classification.
# Features Used
  Hours studied
  Attendance percentage
  Previous grade
# Models Implemented
  Logistic Regression
  Decision Tree Classifier
# Evaluation Metrics
  Accuracy
  Confusion Matrix
  Feature Importance (Decision Tree)
# Key Learnings
  Importance of train-test splitting
  Feature scaling for model stability
  Comparing model performance
  Interpreting classification results
  
  One insight observed was that previous grade had the strongest predictive impact compared to attendance and study hours.
